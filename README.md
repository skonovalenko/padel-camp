# Padel Camp Mallorca

Landing page for a padel sports camp in Mallorca, built for the Ukrainian community abroad.

## Stack

- [Astro](https://astro.build/) v4 — static site generator
- [Tailwind CSS](https://tailwindcss.com/) v3 — styling
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build   # output to dist/
npm run preview # preview the build locally
```

## Deployment

Deployed to Vercel: **https://padel-camp.vercel.app/**
